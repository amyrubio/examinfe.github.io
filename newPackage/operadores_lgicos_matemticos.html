Operadores lógicos matemáticos

Operadores en java

En java se utilizan diferentes operadores entre ellos estan los operadores matemáticos.

Son: +,*,-,/

Se utilizan para realizar operaciones entre las variables en java, Algunos ejemplos serían para realizar un cálculadora solamente de sumas dond se emplearían estos operadores (+).

Operadores relacionales

Los operadores relacionales determinan si una condición es verdadera o falsa y también para tomar una desición.
